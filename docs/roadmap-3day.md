# E.O.M 3-Day Roadmap

## 목표

3일 안에 로컬에서 주요 기능이 동작하고, Render 무료 티어에 배포 가능한 Spring Boot MVC 앱을 완성합니다.

## D1: 구조와 인증

목표:
- Spring Boot 구조 안정화
- 로그인/관리자 흐름 완성
- 문서 기준 확정

작업:
- 프로젝트 구조 점검 완료
- PRD, 아키텍처, 디자인 가이드, Git 가이드 작성 완료
- 사용자/관리자 로그인 기본 흐름 완료
- 관리자 페이지 기본 조회 완료
- H2 시드 데이터 완료
- 라이트/다크 토글 기본 연결 완료
- React index/login 시안의 Spring Thymeleaf 이식 완료

완료 기준:
- `/`, `/login`, `/dashboard`, `/admin` 이동 확인
- 관리자 계정으로 `/admin` 접근 가능
- 테스트 통과

## D2: 커뮤니티 핵심 기능

목표:
- SHOW/CAST/HYPE/LINK 게시판 탐색
- 검색, Events, 마이페이지와 인스타그램 링크

작업:
- CommunityController, CommunityService 구현
- 게시글 목록/상세
- `/posts?q=` 통합 검색과 `/posts?tag=` 태그 검색
- 빈 검색어 전체 목록 노출 방지
- HYPE 관리자 승인 행사 필터 탐색
- `/dancers` 댄서 탐색과 작성자 프로필 이동
- 마이페이지
- 프로필/계정 수정
- 포트폴리오 선택과 최대 3개 상단 고정
- Joined Events 추가/수정/삭제
- 프로필 인스타그램 링크
- 게시글 인스타그램 릴스/게시물 URL 필드
- 유튜브 URL 또는 외부 영상 URL 필드
- `MediaType` 기반 미디어 구분
- 목록용 `thumbnailUrl` 카드 이미지
- 공개 게시물 바로보기 영역 또는 링크 폴백 확장 구조

완료 기준:
- 4개 보드에서 목록/상세 조회 가능
- 검색과 태그 탐색이 같은 결과 화면에서 동작
- HYPE 관리자 승인 행사 필터 확인 가능
- 사용자별 활동 확인 가능
- 인스타그램/영상 URL이 화면에 표시됨
- 직접 영상 업로드와 직접 이미지 업로드는 MVP 범위에서 제외됨
- 게시글 작성 폼은 `/posts/new`에서 URL 기반 미디어와 Live Preview 방식으로 구현됨. 보드 목록 `New Post`는 현재 보드를 작성 탭에 자동 적용함.

## D3: 디자인과 배포 준비

목표:
- 기존 index/login 시안 이식
- 대시보드 이후 화면을 스트릿/어워즈 감성으로 정리
- Render 배포 준비

작업:
- index/login 디자인 정교화 완료
- 대시보드 이미지 카드 구성
- 대시보드 미디어/릴스 중심 SHOW 카드 구성
- 보드 목록 카드형 디자인
- 관리자 페이지 가독성 개선
- 이미지 선별/압축
- Render 환경변수 문서 확인
- 최종 테스트

완료 기준:
- 로컬에서 발표 가능한 시나리오 완성
- Render 배포 가능한 빌드 상태
- README와 docs가 실제 구현과 맞음

## 버퍼

- 오류 수정
- 화면 캡처
- 발표 시나리오 정리
- Render 배포 확인
